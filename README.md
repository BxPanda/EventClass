# EventClass
Event Class , task from university, using algorithms

Завдання :

Розробити тип для задання інформації про події за такими параметрами : дата,опис,категорія,місце,список прізвищ учасників.
Дані про події подано кількома окремими файлами. Отримати :

1. Впорядкований за датами,місцем і категрією повний перелік усіх подій.
2. Для кожної наявної категорії перелік подій за заданий період часу.
3. Рейтинг осіб,які фігурують у подіях в заданому місці.(Чим більше раз присутній , тим більший рейтинг)


Task :

Develop a type for specifying information about events by the following parameters: date, description, category, place, list of names of participants.
Event data is presented in several separate files. Get:

1. A complete list of all events is sorted by date, place and category.
2. For each available category, a list of events for a given period of time.
3. Rating of people who appear in the events in a given place. (The more times present, the higher the rating)
